# Alzahra Transportation System

A comprehensive transportation booking and fleet management platform designed to facilitate passenger transportation services, vehicle reservations, and customer management.

## Features

* User Registration and Authentication
* User Profile Management
* Vehicle Reservation System
* Fleet Management
* Reservation Tracking
* Payment Management
* Customer Messages Management
* Admin Dashboard
* Arabic and English Language Support
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap
* Axios
* JSON

## Modules

* Home Page
* Vehicle Listing
* Booking System
* User Management
* Dashboard
* Payments
* Reservations
* Profile Management
